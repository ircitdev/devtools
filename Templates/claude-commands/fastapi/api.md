description: Создание нового API endpoint (FastAPI)
---

Создай новый REST API endpoint для FastAPI.

Включи:
- Router definition
- Request/Response Pydantic models
- Path parameters и query parameters с validation
- HTTP methods (GET, POST, PUT, DELETE)
- Error handling (HTTPException)
- Docstring и OpenAPI documentation
- Dependency injection где уместно
- Status codes
- Response model
- Тесты для endpoint

Следуй RESTful принципам и FastAPI best practices.
