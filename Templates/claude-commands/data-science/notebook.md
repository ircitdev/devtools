Создай Jupyter notebook для анализа данных.

Структура notebook:
1. **Setup** - импорты и конфигурация
2. **Load Data** - загрузка данных
3. **Explore Data** - первичный анализ (head, info, describe)
4. **Clean Data** - обработка пропусков, дубликатов
5. **Transform Data** - feature engineering
6. **Visualize** - графики и визуализация
7. **Analyze** - статистический анализ
8. **Conclusions** - выводы

Используй:
- pandas для данных
- matplotlib/seaborn для визуализации
- Понятные markdown заголовки
- Комментарии в коде
- Выводы после каждой секции
